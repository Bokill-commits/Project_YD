# 🎬 Project_YD | 영상 데이터 수집 자동화 스크립트

학습할 용도로 영상 데이터의 수집이 필요하여 Google Colab 환경에서 yt-dlp를 활용해
유튜브 영상을 원하는 화질로 다운로드할 수 있는 자동화 스크립트를 구현하였습니다.

---

## 📌 프로젝트 특징

- 🎥 유튜브 영상 정보 자동 분석
- 📊 선택 가능한 화질 리스트 제공
- ⏳ 다운로드 진행률 실시간 출력
- 💾 선택한 화질로 자동 다운로드
- 📁 PC로 바로 다운로드 지원 (Google Colab)

---

## 🛠️ 기술 스택

| Language | Tools | Core |
|-----------|------|------|
| Python | Google Colab | yt-dlp, ffmpeg |
