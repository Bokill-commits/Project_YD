## 📌 프로젝트 특징

Google Colab 환경에서 yt-dlp를 활용해 유튜브 영상을 원하는 화질로 수집할 수 있는 자동화 스크립트를 구현하였습니다.

- 🎥 유튜브 영상 정보 자동 분석
- 📊 선택 가능한 화질 리스트 제공
- ⏳ 다운로드 진행률 실시간 출력
- 💾 선택한 화질로 자동 다운로드
- 📁 PC로 바로 다운로드 지원 (Google Colab)

## ⚙️ 실행 방법

1. Google Colab 실행
2. 라이브러리 환경 설치
3. 메인 코드 실행
4. 데이터 수집 할 유튜브 URL 입력
5. 원하는 화질 선택 → 자동 다운로드
